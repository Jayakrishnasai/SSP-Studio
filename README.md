# SSP Studio

SSP Studio is a premium, cinematic creative studio website designed to showcase high-end visual production services. Built with a focus on high-performance animations and a "pixel-perfect" aesthetic, it provides an immersive experience for clients looking for professional photography and videography.

## 🎬 Cinematic Experience

The core of the website is a custom-built, scroll-triggered frame sequence that creates a cinematic journey as the user explores the site. Every transition is meticulously timed using GSAP and Framer Motion to ensure a premium feel.

## 🚀 Technologies Used

- **Next.js 15**: The latest React framework for high-performance web applications.
- **Tailwind CSS 4**: A modern, utility-first CSS framework for rapid styling.
- **GSAP (GreenSock Animation Platform)**: Used for complex, high-performance scroll-driven animations and canvas rendering.
- **Framer Motion**: Powering smooth, declarative component animations and page transitions.
- **Lenis**: For smooth, high-quality scrolling across all browsers.
- **React Icons**: A comprehensive set of high-quality icons.

## ✨ Key Features

- **Frame Sequence Animation**: A high-resolution canvas-based animation that scrubs through 270+ frames based on scroll position.
- **Smooth Scrolling**: Integrated with Lenis for a fluid, luxurious browsing experience.
- **Interactive Portfolio**: A filtered gallery showcasing projects across various categories (Photography, Videography, Commercial, etc.).
- **Dynamic Services Section**: Modern glassmorphic cards highlighting production services.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices.
- **Modern UI/UX**: Features glassmorphism, cinematic gradients, and subtle noise overlays for a premium studio aesthetic.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.18 or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Jayakrishnasai/SSP-Studio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `/pages`: Next.js Pages router implementation.
- `/components`: Reusable UI components built with React and Framer Motion.
- `/public`: Static assets, including the high-resolution frame sequences.
- `/styles`: Global CSS and Tailwind configuration.
- `/lib`: Utility functions and animation helpers.

## 🌐 Deployment

The project is optimized for deployment on **Vercel**. 

- **Node.js 20** is recommended for best compatibility with Next.js 15.
- **Image Optimization**: Large frame sequences are served as unoptimized static assets to ensure zero-latency rendering on canvas.
